# VoltGuard — Smart Campus Power Monitoring

VoltGuard is a real-time power incident tracking and prediction system for campuses, enabling proactive maintenance and reduced downtime.

## Features
- Real-time reporting for microblackouts, voltage fluctuations, and outages
- Pattern-based risk prediction and early alerts
- Live dashboard with location-based tracking
- Technician assignment and SLA monitoring

## Tech Stack
Frontend: React, Tailwind CSS, Vite  
Backend: Node.js, Express, REST API
